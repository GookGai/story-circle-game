# 🎲 Story Circle

Story Circle เป็นโปรเจกต์เกมกระดานวงเหล้า (Real-time Multiplayer) ที่ออกแบบให้เล่นผ่านมือถือได้อย่างลื่นไหลและสนุกสนาน มีระบบสร้างห้องและเชิญเพื่อนเข้ามาจอยเพื่อร่วมสนุกไปกับเกมต่างๆ

## 📁 โครงสร้างโปรเจกต์ (Monorepo)

- **`backend/`**: เซิร์ฟเวอร์ Node.js + Express, ฐานข้อมูล PostgreSQL (Prisma), และระบบ Real-time ด้วย Socket.IO
- **`frontend/`**: หน้าเว็บ React (Vite) + Socket.IO Client พร้อม UI แบบ Neon Dark Party Theme

## ✨ ฟีเจอร์เด่นของเกม (Key Features)

### 🏇 เกมแข่งม้าโฉมใหม่ (Premium Horse Race)
- **ระบบจัดลำดับม้าเข้าเส้นชัย 🏁**: แสดงผลลำดับม้าที่วิ่งเข้าเส้นชัยตั้งแต่ **อันดับ 1 ถึงอันดับ 5** อย่างสวยงามในหน้าสรุปผลการแข่ง พร้อมเหรียญรางวัลเกียรติยศ (🥇, 🥈, 🥉)
- **ระบบตารางคะแนนสะสมของผู้เล่น 🏆**: เพิ่มความสนุกให้ลุ้นระทึกกว่าเดิมในการเล่นหลายๆ รอบ โดยมีตารางคะแนนเก็บบันทึกสะสมดาว (Stars ⭐) ให้กับผู้เล่นตามลำดับม้าที่แทงในแต่ละรอบ:
  - ม้าเข้าอันดับที่ 1: **รับ +5 แต้ม** ⭐
  - ม้าเข้าอันดับที่ 2: **รับ +4 แต้ม** ⭐
  - ม้าเข้าอันดับที่ 3: **รับ +3 แต้ม** ⭐
  - ม้าเข้าอันดับที่ 4: **รับ +2 แต้ม** ⭐
  - ม้าเข้าอันดับที่ 5: **รับ +1 แต้ม** ⭐
- **ระบบแสดงดาวความสำเร็จ ⭐**: รายชื่อผู้เล่นใน Lobby หน้ารอห้องหลักจะแสดงสะสมดาว ⭐ นีออนข้างๆ จำนวนแก้วเครื่องดื่ม 🍺 ทันทีเพื่อให้ทุกคนรู้ว่าใครกำลังนำโด่งเป็นแชมเปี้ยนประจำตาราง!
- **การปรับสมดุลม้าให้สูสีที่สุด ⚖️**: ปรับแต่งช่วงสุ่มพลังเริ่มต้นของม้าให้แคบลงเฉลี่ยอยู่ที่ **`70-90`** เพื่อให้การวิ่งเบียดเสียด ลุ้นสนุกจนแทบหยุดหายใจ ชัยชนะจะตัดสินกันที่บัฟพุ่งตัว (Burst) หรือสะดุดล้ม (Stumble) เท่านั้น!

## 🚀 วิธีการรันโปรเจกต์บนเครื่องของคุณ

เนื่องจากโปรเจกต์นี้แยกเป็น 2 ส่วน คุณต้องรันทั้งเซิร์ฟเวอร์ Backend และหน้าเว็บ Frontend พร้อมกันครับ

### 1. การรัน Backend

เปิด Terminal ของคุณแล้วเข้าไปที่โฟลเดอร์ `backend`:
```bash
cd backend
npm install
```

**การตั้งค่าฐานข้อมูล:**
โปรเจกต์นี้ใช้ PostgreSQL ตรวจสอบให้แน่ใจว่าคุณได้รัน PostgreSQL บนเครื่องของคุณแล้ว 
เปิดไฟล์ `backend/.env` และแก้ไข `DATABASE_URL` ให้ตรงกับระบบของคุณ ตัวอย่างเช่น:
```env
DATABASE_URL="postgresql://postgres:รหัสผ่านของคุณ@localhost:5432/storycircle?schema=public"
```

สร้างตารางและรันเซิร์ฟเวอร์:
```bash
npx prisma db push
npm run dev
```
เซิร์ฟเวอร์จะรันอยู่ที่ `http://localhost:3001`

### 2. การรัน Frontend

เปิด Terminal ใหม่ (อีกแท็บหนึ่ง) แล้วเข้าไปที่โฟลเดอร์ `frontend`:
```bash
cd frontend
npm install
npm run dev
```
หน้าเว็บจะเปิดให้ใช้งานที่ `http://localhost:5173`

## 🎮 วิธีการเล่น

1. เข้าไปที่ `http://localhost:5173` ผ่านเบราว์เซอร์
2. กรอกชื่อผู้ใช้และเลือกรหัสผ่านเพื่อเข้าสู่ระบบ (สามารถเลือกรูปโปรไฟล์สัตว์สุดกวนได้)
3. กด **"สร้างห้อง"** เลือกเกมที่ต้องการเล่น (เช่น เกมแข่งม้า) และกำหนดจำนวนคน
4. คุณจะได้ **รหัสห้อง 6 หลัก** มา ให้เพื่อนเอาไปกรอกในหน้าแรกเพื่อจอยห้องเดียวกัน
5. เมื่อคนครบแล้ว เจ้าของห้องกด **"เริ่มเกม"** ลุยได้เลย!

## ☁️ การนำขึ้นเซิร์ฟเวอร์จริง (Deploy บน Railway.app)

โปรเจกต์นี้เตรียมโค้ดรองรับการนำขึ้นระบบง่ายๆ ผ่าน Railway ไว้แล้ว:

1. ดันโค้ดทั้งหมด (ทั้ง `frontend` และ `backend`) ขึ้น GitHub Repository
2. สมัครและล็อกอินเข้า Railway.app
3. กด **New Project** -> **Deploy from GitHub repo**
4. เพิ่มบริการ **PostgreSQL** ใน Railway เพื่อใช้งานฐานข้อมูล
5. สร้าง Service แยก 2 ตัว สำหรับ Backend และ Frontend:

**ตั้งค่า Backend Service:**
- **Root Directory**: `backend`
- **Build Command**: `npm install && npx prisma generate`
- **Start Command**: `npm start`
- **Environment Variables**:
  - `DATABASE_URL`: นำมาจากค่าของ PostgreSQL ใน Railway
  - `JWT_SECRET`: รหัสลับสำหรับ Token (ตั้งเป็นอะไรก็ได้)
  - `PORT`: ปล่อยเป็นค่าเริ่มต้น (Railway จะจัดการให้)

**ตั้งค่า Frontend Service:**
- **Root Directory**: `frontend`
- **Build Command**: `npm run build`
- **Start Command**: `npm run preview` (หรือใช้ Static Hosting แบบ Vite)
- **Environment Variables**:
  - `VITE_API_URL`: URL ของ Backend ที่ได้จากข้อด้านบน
