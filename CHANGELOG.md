# 📝 Changelog (ประวัติการอัปเดต)

เอกสารนี้จะบันทึกประวัติการเปลี่ยนแปลงและฟีเจอร์ใหม่ที่เพิ่มเข้ามาในแอปพลิเคชัน Story Circle

## [v1.0.3] - เพิ่มมินิเกมที่ 3 (Panic Jump) และปรับจูนระบบ

### 🚀 Features (มีอะไรใหม่)
- **เพิ่มมินิเกมที่ 3**: "ร่มชูชีพวัดใจ (Panic Jump: Blind Edition)" เกมแนววัดความกล้าที่ให้ผู้เล่นแย่งกันกดกระโดดภายใน 8 วินาที 
- **ระบบคะแนนตามเวลา (Time-based Scoring)**: ในเกม Panic Jump หากคุณกดช้า (แต่ยังอยู่ในโควต้า) จะได้รับคะแนนสะสมหลักร้อย! เพิ่มความตื่นเต้นและบีบให้ทุกคนต้องกล้าเสี่ยง
- **ตั้งค่าจำนวนรอบ**: ตอนสร้างห้องและในหน้าล็อบบี้ หัวหน้าห้องสามารถตั้งค่าได้แล้วว่าเกม Panic Jump จะเล่นกี่รอบ (1-10 รอบ)
- **แอนิเมชันดิ่งพสุธา**: เพิ่มหน้าจอระหว่างการคำนวณผล ที่จะโชว์แอนิเมชันตัวละครของผู้เล่นทุกคนดิ่งลงมาจากฟ้า (คนรอดร่มชูชีพกาง, คนตายร่มไม่กาง)
- **ตารางคะแนนรวมในเกม**: เพิ่มการแสดงตารางสรุปคะแนนรวมล่าสุดในหน้าเปิดผล (Reveal Phase) ของแต่ละรอบ

### 🐛 Bug Fixes (แก้ไขข้อผิดพลาด)
- **แก้บั๊กระบบค้างเมื่อกดรัวๆ (Double Click Prevention)**: แก้ไขปัญหาที่ผู้เล่นเผลอกดปุ่ม "กระโดด" สองครั้งติดกันแล้วระบบเด้ง Alert กวนใจจนทำให้จับเวลาค้าง
- **แก้บั๊กเซิร์ฟเวอร์หลุด**: เพิ่ม `createdAt` ให้กับตาราง Database เพื่อป้องกันปัญหา backend หยุดชะงัก (Crash) ระหว่างคำนวณผลเวลา
