# 🎲 Story Circle

Story Circle เป็นโปรเจกต์เกมกระดานวงเหล้า (Real-time Multiplayer) ที่ออกแบบให้เล่นผ่านมือถือได้อย่างลื่นไหลและสนุกสนาน มีระบบสร้างห้องและเชิญเพื่อนเข้ามาจอยเพื่อร่วมสนุกไปกับเกมต่างๆ ในวงเหล้าแบบเรียลไทม์

---

## 📁 โครงสร้างโปรเจกต์ (Monorepo)

- **`backend/`**: เซิร์ฟเวอร์ Node.js + Express, ฐานข้อมูล PostgreSQL (Prisma), และระบบ Real-time ด้วย Socket.IO
- **`frontend/`**: หน้าเว็บ React (Vite) + Socket.IO Client พร้อม UI แบบ Neon Dark Party Theme

---

## ✨ ฟีเจอร์เด่นของเกม (Key Features)

### 👥 1. ระบบผู้ใช้ชั่วคราว & การจอยแบบรวดเร็ว (Guest Login & Quick Join)
- **ล็อกอินเฉพาะโฮสต์**: เพื่อความสะดวก มีเพียงเจ้าของห้อง (Host) เท่านั้นที่ต้องลงทะเบียนบัญชี ส่วนเพื่อนผู้เล่นคนอื่นๆ สามารถเข้าร่วมวงได้โดยไม่ต้องสมัครสมาชิก
- **สแกนจอยด้วย QR Code**: หน้าล็อบบี้แสดงปุ่ม QR Code ที่สร้างขึ้นแบบไดนามิกเพื่อให้ผู้เล่นอื่นสแกนเข้าสู่ห้องหรือเปิดลิงก์ตรงได้ทันที
- **สมัคร Guest ทันที**: ผู้เข้าเล่นเพียงแค่พิมพ์ชื่อเล่น (Nickname) และจิ้มเลือกสัตว์อวาตาร์ระบบจะพาเข้าห้องเล่นทันทีในคลิกเดียว

### 🔄 2. การสลับเกมและซิงโครไนซ์ห้อง (Game Swapping & Sync Lobby)
- **สลับเกมกลางห้อง**: โฮสต์สามารถเลือกสลับประเภทเกมระหว่าง **"แข่งม้า"** และ **"โหวตเสียงส่วนน้อย"** ได้โดยตรงจากหน้าล็อบบี้ โดยไม่ต้องกดปิดห้องหรือสร้างรหัสห้องใหม่
- **พาทุกคนกลับล็อบบี้**: เมื่อโฮสต์กดปุ่ม "กลับหน้าห้อง" จากในหน้ารายงานผลเกมใดๆ ระบบจะดึงผู้เล่นทุกคนในห้องย้อนกลับไปที่หน้าล็อบบี้หลักโดยอัตโนมัติ

### 🧹 3. ลบข้อมูลอัตโนมัติ (Zero-History Database)
- **ล้างข้อมูลรอบเก่า**: เพื่อประหยัดพื้นที่ฐานข้อมูล เมื่อระบบเริ่มรอบใหม่ ข้อมูลรอบเดิมในห้องจะถูกทำลายทิ้งทันที
- **ลบห้อง & บัญชีเมื่อร้าง**: เมื่อผู้เล่นคนสุดท้ายออกจากห้องไป ระบบจะลบห้องออกจากฐานข้อมูลทันทีด้วยระบบ Cascade และล้างบัญชี Guest ที่ค้างอยู่เพื่อความสะอาดของตารางฐานข้อมูล

### 🏇 4. เกมแข่งม้าโฉมใหม่ (Premium Horse Race)
- **ระบบกูรูประเมินผลม้าแข่ง**: มี 3 กูรูสุดปั่น (**กรู(ไม่)รู**, **กรูเดา**, และ **กรูมั่ว**) ช่วยให้คำแนะนำและทำนายเกรดดาว (1-5 ดาว) โดยแต่ละรอบกูรูทุกคนจะมีม้าเต็ง 5 ดาว (👑) เสมอ 1 ตัว และม้าบ๊วย 1 ดาว (💩) เสมอ 1 ตัว ส่วนม้าตัวอื่นๆ จะเป็นแบบสุ่ม ซึ่งคำนวณผ่าน Seed ของรอบการเล่นทำให้มั่นใจได้ว่าตัวเลขดาวจะคงที่และยุติธรรมสำหรับทุกคน
- **ซ่อนสเตตัสความเร็ว**: ก่อนการแข่งจะมีการปกปิดความเร็วจริงของม้าเพื่อให้ผู้เล่นต้องเดิมพันผ่านบทวิเคราะห์ของกูรูเท่านั้น และจะเฉลยสเตตัสจริงหลังจากวิ่งเข้าเส้นชัยเพื่อความขำขันและบันเทิง
- **การจัดอันดับและคะแนนสะสม**: สรุปผลความสำเร็จอันดับ 1-5 พร้อมเหรียญเกียรติยศ และเก็บคะแนนสะสมดาว (Stars ⭐) ให้ผู้เล่นแต่ละคนเพื่อโชว์ในหน้าตารางคะแนนและหน้าล็อบบี้

### 🗳️ 5. เกมเสียงส่วนน้อย (Minority Vote)
- **คิดโจทย์ผ่านพลัง AI**: ผู้ตั้งคำถามสามารถพิมพ์เรื่องเอง หรือให้ AI ช่วยสุ่มแต่งโจทย์จาก OpenRouter API (Gemini-2.5-Flash) ที่มีความกวนๆ และเข้ากับวงเหล้าตาม 6 หมวดหมู่ โดยมีระบบคำถามออฟไลน์แบบคละหมวดหมู่รองรับกรณีอินเทอร์เน็ตมีปัญหา
- **ชาร์ตรายงานผลแบบซูมได้ (Zoomable Chart)**: กราฟแท่งสีสันนีออนสรุปสถิติโหวตที่ผู้เล่นสามารถเลื่อน Slider ปรับขนาดการซูม (Zoom 1x - 3x) เพื่อให้สามารถสแกนอ่านรายชื่อผู้โหวตแต่ละข้างบนหน้าจอโทรศัพท์มือถือได้อย่างสะดวก
